Unified Architecture Graph

Every API route, Lambda, the Step Functions parallel fan-out, and all backend services connected end to end — read left → right.

Amber edges = asynchronous, non-blocking flow. deleteOrder fires StartExecution and returns immediately; the Step Functions Parallel state then runs branch A (SNS publish) and branch B (S3 backup) at the same time.

1 CRUD path

UI → API Gateway → Lambda (createOrder / getAllOrders / getOrder / updateOrder) → DynamoDB. getAllOrders queries the gsi-by-date GSI for creation-date sorting.

2 Delete fan-out (event-driven core)

deleteOrder removes the item → StartExecution → returns 200 immediately. Step Functions Parallel: Branch A SNS email + Branch B S3 TXT backup. Delete never waits for either.

3 Notification subscription

UI → /subscriptions (subscribe) / /unsubscribe → SNS Subscribe/Unsubscribe. Registered emails then receive delete notifications with no manual action.

4 PDF summary download

UI → /reports/deleted-pdf → generatePdfSummary lists all S3 TXT backups, builds a summary PDF, stores it in S3, and returns the download URL in the response body.

5 Image analysis (freestyle)

UI → /analyze-image → analyzeImage Lambda → Amazon Rekognition DetectLabels → detected labels returned and suggested as the order description in the UI.

How the services integrate

The Amplify-hosted web client is pure presentation — it only makes HTTPS calls to API Gateway, which routes each request to a dedicated Lambda function holding all business logic. Order data persists in a single DynamoDB orders table; a Global Secondary Index lets us return all orders sorted by creation date efficiently.

Deletion is the event-driven heart of the system. deleteOrder removes the record and immediately hands off to a Step Functions state machine via a non-blocking StartExecution call, so the user gets an instant response. The state machine's Parallel state simultaneously (A) publishes to SNS — emailing every subscriber the deleted order's details — and (B) writes a TXT backup to S3. Neither branch delays the delete.

Subscriptions are handled directly against the SNS topic through dedicated APIs. The PDF summary API aggregates every TXT backup in S3 into a single PDF stored back in S3, returning a download URL. The freestyle Amazon Rekognition integration analyzes an uploaded product photo and suggests the order description in the UI. All compute runs under the Learner Lab LabRole.
